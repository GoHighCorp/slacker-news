Slacker News is an integration for Slack that provides trending news and articles.

It currently supports the BBC, FiveThirtyEight, Hacker News, Product Hunt, and Vice.

INSTALLATION INSTRUCTIONS

First, set up a new slash command.

Select something memorable like '/news' and then click 'Add Slash Command Integration'

Next, change the URL to be 'slckr-nws.herokuapp.com'

Lastly, change the method to be 'GET'

Voila! All ready to go. Now get on with checking Hacker News in your team chat.
